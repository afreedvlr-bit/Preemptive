# Preemptive# Preemptive Priority CPU Scheduling

This project implements Preemptive Priority Scheduling in C.

## 📌 Description
- Higher priority value means higher priority.
- CPU always selects the process with the highest priority.
- Preemption is allowed.

## 📥 Input Format
n  
PID AT BT Priority  

## 📤 Output
- Waiting Time
- Turnaround Time
- Average Waiting Time
- Average Turnaround Time

## ▶️ How to Run

```bash
gcc priority.c -o priority
./priority
